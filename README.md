# The Magician

A modern text adventure RPG based on Raymond E. Feist's *Magician: Apprentice* and *Magician: Master* from the Riftwar Saga.

**[Play Now](https://NORS3AI.github.io/The-Magician/)** | [Features](features.md) | [Roadmap](todo.md)

## Overview

## Tech Stack

- **Vite** - Lightning-fast build tool
- **Tailwind CSS v4** - Modern utility-first styling
- **TypeScript** - Type-safe game logic
- **GitHub Pages** - Free static hosting
- **localStorage** - Persistent game saves

## Features

- **Two Paths** - Play as Tomas (Warrior) or Pug (Mage)
- **RPG Stats** - Strength, Constitution, Agility, Intelligence, Willpower, Charisma
- **Combat System** - Turn-based tactical battles
- **Magic System** - Spells, mana, and arcane abilities
- **Inventory** - Equipment, items, and resource management
- **Story-Driven** - Branching narrative faithful to Feist's world
- **Mobile-Ready** - Responsive design for any device
- **Dark Theme** - Beautiful, immersive UI

## Quick Start

### Play Online
Visit **[https://NORS3AI.github.io/The-Magician/](https://NORS3AI.github.io/The-Magician/)**

### Run Locally
```bash
git clone https://github.com/NORS3AI/The-Magician.git
cd The-Magician
npm install
npm run dev
```
Open [http://localhost:5173](http://localhost:5173)

### Build for Production
```bash
npm run build
npm run preview
```
