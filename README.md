# The-Magician

A text adventure RPG based on Raymond E. Feist's *Magician: Apprentice* and *Magician: Master* from the Riftwar Saga.

## Overview

Follow the journeys of Tomas and Pug, two boys from Crydee whose destinies will shape the fate of two worlds. Choose your path—the Warrior or the Mage—and experience the epic story through your character's eyes.

## Play Now

```bash
# Clone the repository
git clone https://github.com/NORS3AI/The-Magician.git
cd The-Magician

# Set up virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the game (CLI)
python main.py

# OR run the web interface
python app.py
# Opens at http://localhost:5000
```

## About the Game

- **Genre:** Text Adventure RPG
- **Platform:** Python (command-line)
- **Paths:** Two distinct storylines (Tomas or Pug)
- **Gameplay:** RPG stats, combat, magic, inventory, and survival mechanics

The game stays true to Feist's world and narrative while giving players agency through their choices and actions.
